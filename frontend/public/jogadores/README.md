# Artes Temporárias dos Jogadores

Este diretório contém as artes temporárias dos jogadores por raridade.

As imagens devem seguir o padrão:
- `normal-1.png` até `normal-5.png` (60% de chance)
- `raro-1.png` até `raro-5.png` (30% de chance)
- `epico-1.png` até `epico-5.png` (8% de chance)
- `lendario-1.png` até `lendario-5.png` (2% de chance)

Por enquanto, o sistema usa emojis como placeholder. Você pode substituir essas imagens posteriormente.
